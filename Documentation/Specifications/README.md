# Specifications

This directory contains copies of the Gentoo **Package Manager Specification**
(PMS) for EAPI versions 2 through 9. These documents are the authoritative
reference for the ebuild repository format and package manager behaviour that
portage-ng implements.

| File       | EAPI | Online                                              |
|------------|------|-----------------------------------------------------|
| `pms-2.pdf` | 2  | [PMS 2](https://projects.gentoo.org/pms/2/pms.html) |
| `pms-3.pdf` | 3  | [PMS 3](https://projects.gentoo.org/pms/3/pms.html) |
| `pms-4.pdf` | 4  | [PMS 4](https://projects.gentoo.org/pms/4/pms.html) |
| `pms-5.pdf` | 5  | [PMS 5](https://projects.gentoo.org/pms/5/pms.html) |
| `pms-6.pdf` | 6  | [PMS 6](https://projects.gentoo.org/pms/6/pms.html) |
| `pms-7.pdf` | 7  | [PMS 7](https://projects.gentoo.org/pms/7/pms.html) |
| `pms-8.pdf` | 8  | [PMS 8](https://projects.gentoo.org/pms/8/pms.html) |
| `pms-9.pdf` | 9  | [PMS 9](https://projects.gentoo.org/pms/9/pms.html) |
