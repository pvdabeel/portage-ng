# Graph assets

Stylesheets and fonts used by the portage-ng `--graph` HTML output.

## Stylesheets/

Stylesheets loaded by the generated HTML pages.

| File             | Installed as       | Purpose                                                                          |
|------------------|--------------------|----------------------------------------------------------------------------------|
| `portage-ng.css` | `.portage-ng.css`  | Unified stylesheet for all graph pages (deptree, gantt, detail, terminal, index)  |
| `proof.css`      | `.proof.css`       | Proof-tree page styles (used by print-aha scripts)                               |
| `index.css`      | `.index.css`       | Index page styles                                                                |

## Fonts/

Custom web fonts bundled for offline use by graph pages.

| File        | Installed as  | Purpose                         |
|-------------|---------------|----------------------------------|
| `meslo.ttf` | `.meslo.ttf`  | Meslo font for proof rendering   |
