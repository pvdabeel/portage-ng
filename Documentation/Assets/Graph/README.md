# Graph assets

Stylesheets used by the portage-ng `--graph` HTML output.

## Stylesheets/

Page-specific stylesheets loaded by the generated HTML pages.

| File              | Purpose                                                  |
|-------------------|----------------------------------------------------------|
| `portage-ng.css`  | Unified stylesheet for all graph pages (deptree, gantt, detail, terminal, index) |
| `.proof.css`      | Legacy proof-tree page styles                            |
| `.index.css`      | Legacy index page styles                                 |

## Fonts/

Custom web fonts bundled for offline use by graph pages.
